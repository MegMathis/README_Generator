## Installation

1. npm init
2. npm i iquirer

## License

MIT

## Usage

1. Run `node index.js`
2. Answer each prompt
3. License - use a drop down box to select your license.
4. Once finished you will be shown "README is generated"
5. Right click on README.md and click "Open in Preview"
